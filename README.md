# Compañeros PI

Proyecto académico de desarrollo web realizado por estudiantes del
Politécnico Internacional.

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5

## Componentes de Bootstrap implementados
- Navbar (`navbar`, `navbar-expand-lg`, `fixed-top`)
- Sistema de grillas (`container`, `row`, `col-*`)
- Cards (`card`) para la sección de servicios
- Botones (`btn`, `btn-primary`, `btn-outline-*`)
- Formularios (`form-control`, `form-label`)
- Utilidades de espaciado (`py-5`, `mb-*`, `text-center`)
- Comportamiento responsive (`collapse`, `navbar-toggler`)

## Estructura principal
```
├── index.html          # Página principal con hero y resumen de servicios
├── servicios.html      # Detalle de servicios con diseño alternado
├── contactenos.html    # Formulario de contacto y datos institucionales
├── styles.css          # Ajustes específicos sobre Bootstrap
└── img/                # Activos gráficos usados en las secciones
```

## Cómo ejecutar en local
1. Clona el repositorio.
2. Abre la carpeta en tu navegador preferido (doble clic en `index.html`).
3. Para ver los demás apartados usa la barra de navegación superior.
